Introducción a HTML

Los documentos HTML, se basan en una estructura jerarquica de etiquetas (HyperText Markup Language)

Cada página web HTML, tiene una estructura básica, comienzan con !DOCTYPE html (esto le identifica a los navegadores el tipo de archivo que tiene que enseñar), y además usan las etiquetas de head (encabezado), html (la etiqueta que agrupa la web) y body (que es el contenido de la web).

<!DOCTYPE html>

Etiquetas básicas

<title> El título de la ventana en navegador
<h1 a h6> Definen los títulos y subtitulos, de H1 a H6
<p> Define un párrafo de text (paragraph)
<p> Define una celda, en una fila (table data)
<br> Define una celda, en una fila (table data)

Atributos HTML

Son elementos que añaden información extra a las etiquetas HTML, ya sea para modificar su comportamiento o apariencia. Siguen el formato: nombre="valor

Texto formateado

Son etiquetas que se pueden añadir al texto para modificarlo visualmente, ya sea por importancia del texto, estética o comodidad.

<b> y <strong> pone el texto en negrita, b no aplica importancia semántica y strong sí, indica que el texto es importante.
<i> y <em> Aplica cursiva, i no aplica importancia y em sí.
<u>Subrayado Subralla el texto.
<strike> y <s> Tacha el texto, se usa para borrar o restar importancia.
<sup> Superíndice Eleva el texto, se usa para operaciones matemáticas. 23
<sub> Subíndice Baja el texto, se usa en ecuaciones químicas. H2O

Insercción de imágenes en HTML

La etiqueta para añadir imágens es "<img>", esta etiqueta no necesita cerrarse.
Es recomendable añadir un texto de descripción de la imagen, por motivos de accesibilidad y SEO. Ejemplo de etiqueta img:
<img src="URL o Ruta" alt="texto_descriptivo" style="widht: 100%">

En este ejemplo, src sirve para señalar la fuente (source) de la imagen, puede ser una ruta local relativa o absoluta(no recomendable), o una url de internet. Por otra parte, alt marca la descripcion de la imagen, esta aparece si la imagen no carga. Por último widht y height se usan para delimitar el tamaño de la imagen, es recomendable usar porcentajes y no valores fijos (en pixeles), para mejorar la visibilidad en otros dispositivos.


Inserción de enlaces en HTML

La etiqueta <a> se usa para crear enlaces a otras páginas web, documentos o secciones dentro de la misma página. El atributo principal es href, que define la URL de destino del enlace.

Otro atributo importante es target, que especifica dónde se abrirá el enlace. Los valores más comunes son:

  • _self: Abre el enlace en la misma pestaña (valor por defecto).
  • _blank: Abre el enlace en una nueva pestaña.

Listas en HTML

Las listas se utilizan para agrupar un conjunto de elementos relacionados.

Listas No Ordenadas (<ul>)

Se utilizan para listas de elementos sin un orden específico. Los elementos se muestran con viñetas. Cada elemento de la lista se define con la etiqueta <li> (list item).

Ejemplo:

  • Primer item
  • Segundo item
  • Tercer item

Listas Ordenadas (<ol>)

Se utilizan para listas en las que el orden de los elementos es relevante. Los elementos se numeran automáticamente (1, 2, 3, etc.). Cada elemento también se define con la etiqueta <li>.

Ejemplo:

  1. Primer item
  2. Segundo item
  3. Tercer item

Atributos de Listas Ordenadas

Se pueden usar atributos para personalizar las listas ordenadas:

  • type: Define el tipo de numeración de la lista ('1' para números, 'A' para mayúsculas, 'a' para minúsculas, 'I' para números romanos en mayúsculas, 'i' para números romanos en minúsculas).
  • start: Define el número o letra por el que debe comenzar la lista.

Ejemplo de lista con letras mayúsculas:

  1. Primer item
  2. Segundo item

Ejemplo de lista que empieza en el número 5:

  1. Item número 5
  2. Item número 6

Tablas HTML

Las tablas se usan para mostrar datos en formato de filas y columnas.

Etiquetas básicas de tabla:

  • <table>: Define la tabla y actúa como el contenedor principal.
  • <tr>: Define una fila dentro de la tabla (table row).
  • <td>: Crea una celda dentro de una fila (table data).

Etiquetas de estructura de tabla:

  • <thead>: Agrupa el contenido del encabezado de la tabla.
  • <tbody>: Agrupa el contenido del cuerpo de la tabla.
  • <th>: Se utiliza para crear una celda de encabezado (table header). El texto dentro de <th> aparece en negrita y centrado por defecto.

Atributos de Tabla:

  • border: Añade un borde a la tabla y a las celdas.
  • cellpadding: Define el espacio entre el contenido de la celda y el borde de la celda.
  • cellspacing: Controla el espacio entre las celdas de la tabla.
